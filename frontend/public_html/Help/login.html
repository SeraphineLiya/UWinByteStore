Help & User Guide: Login

Before attempting to login, please ensure that you have requested an account using the public sign-up page. If you have created an account, you can enter the email address you used to sign up in the 'username' field and your password in the password field. Both fields are case-sensitive and sensitive to blank spaces (such as a space ' ') at the end of either your password or email, so please be sure that these have not been added inadvertantly if they were not present in your original sign up information.


If you enter a correct username and password combination, you will be shown a brief status message before being redirected to the appropriate location. If admin access has not been granted, you will be redirected to the main home page, while those with admin access will be redirected to the Admin Dashboard. If the only message is 'Login Failed', the most likely cause is that the username and password entered do not match any accounts on file.


If you have forgotten your username, password, or continue to have trouble logging in, please contact one of our administrators.
