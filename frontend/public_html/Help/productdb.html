Help & User Guide: Change Products

Changing products within the front-end admin interface is relatively straightforward. Upon navigating to the relevant page, you will see a button to add a new record, and a list below it with all of the items currently in the database. Click the button to add a new item to the database, or click on any of the existing records to open a page where you can edit them. Before adding or changing any items though, it is important to know how the catalog is structured.

The catalog of items is structured such that each item (record) represents a single option of a product. The individual option by 'Item Name', and all other attributes are the values for that individual item / option. The product name is used to group 1 or more items as a single 'product' in the catalog. Items with matching product names will be displayed as a single product page in the user-facing catalog, with details for the currently selected item / option being shown. For example, I may have a wired mouse as one item and wireless mouse as another. These two entries can have different prices, images, descriptions, and so on, but if they both have the product name "Mouse", they will be grouped under the single title of 'Mouse' on one page with the currently selected option being displayed.

To add a new item / option, simply click the 'Add Item' button. This will open a blank form in which the appropriate values can be entered. If you want this to be a brand new product, simply choose a product name that is not already in use. If you would like this to be a new option for an existing product, enter that product's name (matching case and spacing exactly) and it will be displayed as an option of that product when displayed to the user. The same concepts apply when editing existing items, which you can do by clicking on them in the list.
