Help & User Guide: Sign Up

Currently, signing up for an account is only used to grant access to administrator options. If you need to sign up for an account, please fill in all of the fields on the sign-up page. Fields include first name, last name, email, as well as your password and password confirmation. All fields are mandatory. Once submitting your request for an account, an admin will view your request and assign the appropriate permissions to your account.

If you are unable to submit the sign-up form, please ensure that you have entered valid values for all fields. Name fields are quite flexible, in order to support the wide diversity of names that exist. However, the email field must fulfill the requirements of being a valid email address, including containing an @ symbol, followed by a domain that is theoretically valid. Your password must be at least 8 characters long, and it must match the value in the 'Confirm password' field.

If you continue to have issues signing up for an account, it is possible that there is already an account using the email address you provided. If this is the case, you will see a status message stating this after you attempt to sign up for an account. Limiting your field entries to ASCII characters only when submitting may also be less likely to cause any issues due to how inputs are validated. If you continue to have issues requesting an account, please contact one of our administrators directly.
